# Plant Paradise

## Project Overview

Plant Paradise is a web-based e-commerce application developed for buying and selling plants online. The platform allows users to browse plant collections, add products to their cart, place orders, and manage deliveries efficiently.

## Features

- User Registration and Login
- Plant Catalog Management
- Shopping Cart System
- Order Placement
- Delivery Management
- Database Integration
- Admin Dashboard

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Database

The application stores:

- User Information
- Plant Details
- Cart Data
- Order Details
- Delivery Information

## Modules

### User Module
- Register
- Login
- Browse Plants
- Add to Cart
- Place Orders

### Admin Module
- Manage Plants
- View Orders
- Manage Deliveries
- Manage Users
